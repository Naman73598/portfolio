Web Design

MedNet is a web-based platform that aims to reduce person-to-person contact in medical services by utilizing the internet. It allows patients to book appointments, request organs or register for organ and blood donation. Users can search for hospitals based on their needs. Hospital administrations can use the platform to request critical medical resources from other hospitals, enhancing efficiency and improving patient comfort. MedNet is built using HTML, CSS, JavaScript, Node.js, and MongoDB.
